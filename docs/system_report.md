# FNCO 인플루언서 시딩 플랫폼 — 시스템 보고서

## 1. 개요

FNCO 인플루언서 시딩 플랫폼은 뷰티·패션 브랜드의 인플루언서 마케팅 캠페인을 **기획 → 전략 → 콘텐츠 → 인플루언서 → 실행 → 모니터링**까지 End-to-End로 관리하는 통합 운영 시스템이다.

기존에는 캠페인 생성과 콘텐츠 대시보드 수준의 기능만 존재했으나, **PDA 프레임워크(Persona-Desire-Awareness)** 기반의 전략 설계 체계와 **AI 기반 자동화 기능**을 중심으로 8단계 캠페인 워크플로우가 구축되었다.

---

## 2. 기존 대시보드 기능 (Before)

기존 시스템은 아래 기능을 중심으로 운영되었다.

| 영역 | 기능 | 설명 |
|------|------|------|
| 캠페인 관리 | 캠페인 목록/생성 | 브랜드, 제품, 기간 등 기본 정보로 캠페인 생성 |
| 콘텐츠 대시보드 | 시딩 콘텐츠 조회 | 시딩 콘텐츠를 목록·카드 형태로 열람 |
| 콘텐츠 대시보드 | UGC 콘텐츠 조회 | 인플루언서가 생성한 UGC 콘텐츠 확인 |
| 콘텐츠 대시보드 | 성과 콘텐츠 조회 | 퍼포먼스 기반 콘텐츠 성과 확인 |
| 인플루언서 | 인플루언서 풀 관리 | 인플루언서 기본 정보 및 연락처 관리 |
| 콘텐츠 라이브러리 | 콘텐츠 저장소 | 과거 콘텐츠 아카이빙 및 검색 |

## 3. 확장된 기능 — 관점별 정리

### 3.1 전략 체계화 관점: PDA 프레임워크 도입

기존 캠페인은 직관적 기획에 의존했다. PDA 프레임워크를 도입하여 **타겟(Persona) × 소구점(Desire) × 인지단계(Awareness)** 매트릭스 기반으로 캠페인 전략을 구조화했다.

| 추가 기능 | 설명 |
|-----------|------|
| **페르소나 설정** | 타겟 고객 유형(P2, P7, P9 등)을 프로필·페인포인트·행동 패턴으로 정의 |
| **디자이어 설정** | 제품 소구점(D1, D2, D3)을 감정 트리거와 함께 설계 |
| **인지 단계 매핑** | Problem → Solution → Product → Most Aware (TOFU/MOFU/BOFU) |
| **컨셉 매트릭스** | Persona × Desire × Awareness 조합에서 콘텐츠 컨셉 15개+ 자동 생성 |
| **프로덕트 브리프** | 업로드된 제품 파일(PDF, 이미지)에서 AI가 제품 정보 자동 추출 |

### 3.2 AI 자동화 관점: 전략 생성 및 콘텐츠 제작

사람이 수동으로 수행하던 전략 수립과 크리에이티브 제작 과정에 AI를 접목했다.

| 추가 기능 | 설명 |
|-----------|------|
| **AI 전략 생성** | 채널 배분, 메시징, 타이밍, 예산 전략을 AI가 자동 생성 |
| **내러티브 아크** | Tease → Launch → Spread → LastCall 4단계 콘텐츠 흐름 설계 |
| **AI 이미지 생성** | 컨셉 기반 시딩 이미지 자동 생성 |
| **AI 카피 편집** | 퍼널 단계별 카피(공감, 솔루션, 비교, 교육, 신뢰 등) 자동 작성 |
| **훅 뱅크** | 12종 훅 유형(비주얼 쇼크, 신화, 문화, 페인, 스토리 등) 라이브러리 |
| **시나리오 편집기** | 콘텐츠별 촬영·제작 시나리오 구성 |
| **프로덕션 가이드** | 포맷별 제작 가이드라인 자동 생성 |

### 3.3 인플루언서 매칭 관점: AI 기반 추천 및 파이프라인

인플루언서를 수동으로 탐색·선정하던 방식에서 **AI 추천 + 상태 파이프라인** 관리로 고도화했다.

| 추가 기능 | 설명 |
|-----------|------|
| **AI 인플루언서 매칭** | 페르소나 타겟팅 기반 인플루언서 자동 추천 |
| **딥 프로필 분석** | 팔로워 수, 평균 조회수, 인게이지먼트율 심층 분석 |
| **AI 콘텐츠 플랜** | 인플루언서별 맞춤 콘텐츠 제안 생성 |
| **상태 파이프라인** | Suggested → Selected → Contacted → Confirmed → Posted 5단계 관리 |
| **벌크 관리** | 다수 인플루언서 일괄 선택·상태 변경 |

### 3.4 실행 관리 관점: 아웃리치 및 런칭

기존에는 외부 도구(이메일, 스프레드시트)로 관리하던 실행 과정을 플랫폼 내에 통합했다.

| 추가 기능 | 설명 |
|-----------|------|
| **협업 브리프** | 인플루언서별 맞춤 브리프 자동 생성 |
| **이메일 프리뷰** | 아웃리치 이메일 미리보기 및 발송 |
| **아웃리치 트래킹** | 연락 상태·응답 이력 추적 |
| **채널 셋업** | TikTok, Instagram, YouTube 채널별 배포 설정 |
| **드롭 코디네이션** | 콘텐츠 드롭 타임라인 조율 |
| **스케줄 캘린더** | 게시 일정 캘린더 관리 |
| **승인 플로우** | 런칭 전 최종 승인 프로세스 |

### 3.5 성과 최적화 관점: 실시간 모니터링 및 자동 최적화

캠페인 종료 후 단순 결과 리포팅에서, **실시간 모니터링 + AI 최적화 제안** 체계로 전환했다.

| 추가 기능 | 설명 |
|-----------|------|
| **실시간 KPI 트래킹** | 노출수, 인게이지먼트율, 좋아요 수 실시간 추적 |
| **PDA 히트맵** | Persona × Desire × Awareness 성과를 히트맵으로 시각화 |
| **피로도 트래커** | 오디언스 피로도 감지 및 경고 |
| **얼리 시그널** | 초기 성과 지표 기반 트렌드 예측 |
| **크리에이티브 로테이션** | 저성과 크리에이티브 자동 교체 제안 |
| **채널 리밸런싱** | 채널별 예산 재배분 최적화 |
| **메시지 피봇** | 성과 기반 메시징 전략 변경 제안 |
| **UGC 플라이휠** | UGC 수확 → 큐레이션 → 증폭 자동화 사이클 |

### 3.6 운영 거버넌스 관점: 브랜드 관리 및 감사

조직 단위의 운영 관리·통제 기능을 추가하여 거버넌스를 강화했다.

| 추가 기능 | 설명 |
|-----------|------|
| **Brand DNA 관리** | 미션, 톤앤매너, 비주얼 스타일, 키 메시지 등록 |
| **브랜드 정합성 검증** | 콘텐츠-브랜드 DNA 일치도 자동 검사 |
| **감사 로그** | 모든 변경 이력 자동 기록 |
| **권한 관리** | 역할별 접근 권한 설정 |
| **팀 설정** | 팀 구성 및 역할 관리 |
| **캠페인 템플릿** | 반복 사용 가능한 캠페인 템플릿 갤러리 |
| **내보내기** | PDF/Excel 보고서 출력 |

## 4. 8단계 캠페인 워크플로우 요약

```
Phase 1          Phase 2         Phase 3          Phase 4
캠페인 설정  →   전략 수립   →   콘텐츠 기획  →   크리에이티브 제작
(PDA 정의)      (AI 전략)       (캘린더)         (AI 이미지/카피)
    ↓                                                ↓
Phase 8          Phase 7         Phase 6          Phase 5
모니터링    ←    런칭       ←    아웃리치    ←    인플루언서 매칭
(실시간/최적화)  (드롭 관리)     (브리프/이메일)   (AI 추천)
```

## 6. 결론

FNCO 인플루언서 시딩 플랫폼은 **단순 콘텐츠 대시보드**에서 출발하여, 다음 6가지 관점의 기능 확장을 통해 **인플루언서 마케팅 통합 운영 플랫폼**으로 진화했다.

1. **전략 체계화** — PDA 프레임워크로 감에 의존하던 기획을 데이터 기반 구조로 전환
2. **AI 자동화** — 전략·콘텐츠·매칭 전 과정에 AI를 접목하여 생산성 극대화
3. **인플루언서 매칭 고도화** — 수동 탐색에서 AI 추천 + 파이프라인 관리로 전환
4. **실행 관리 통합** — 외부 도구 의존 없이 아웃리치~런칭까지 플랫폼 내 완결
5. **성과 최적화** — 결과 리포팅에서 실시간 모니터링 + AI 최적화로 고도화
6. **운영 거버넌스** — 브랜드 DNA, 감사 로그, 권한 관리로 조직 통제력 확보
